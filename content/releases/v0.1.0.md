## Highlights

- Established typed Nuxt Content collections for long-form articles and field notes.
- Added searchable indexes, chronological archives, RSS output, and static-friendly routes.
- Completed the pixel-terminal interface across desktop and mobile layouts.

## Improvements

- Search now includes Markdown body sections and keeps its query in the URL.
- Dynamic Open Graph images and Article structured data are generated for published writing.
- The Zpix font is delivered as compressed WOFF2 to reduce transfer size.

## Fixes

- Draft content can no longer be opened directly through a guessed URL.
- Build and typecheck output are isolated from the active development server.
- Empty states, keyboard navigation, focus styles, and reduced-motion behavior are covered.

This release marks the first dependable writing loop: add Markdown, validate it, publish it, and find it again.
::
